# Anomaly/Fraud Detection
- This repository contains my experiments on anomaly detection using deep learning frameworks.
- The first notebook explains how the long short term memory (LSTM) autoencoder model can be used to detect rare events such as machine failure. The NASA C-MAPSS dataset is used for the experimentation.
- The second notebook deals with fraud detection on the PaySim dataset. The classical machine learning model XGBoost, and the deep neural network (DNN) are implemented with behavioural features extracted using SQLite.
- The third notebook uses Pandas for behavioural pattern extraction and an LSTM for fraud detection on the BankSim dataset.
- The fourth notebook performs fraud detection on the Elliptic Bitcoin dataset using a graph neural network (GNN).
